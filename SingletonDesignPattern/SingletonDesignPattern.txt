Singleton Design Pattern
- Singleton là Design Pattern thuộc nhóm Creational (khởi tạo) dùng để đảm bảo:
  Một class chỉ có duy nhất 1 object
  Và cung cấp điểm truy cập toàn cục (global access) tới object đó


Những trường hợp sử dụng Singleton Design Pattern
- Chỉ cần 1 instance duy nhất
- Object đó được dùng ở nhiều nơi
✔ Ví dụ:
Logger
Cache manager
Config system
Thread pool
Connection manager

🔵 Sơ đồ Singleton Design Pattern
              (global access)

            [ Singleton Object ]

🔄 Luồng hoạt động
Client 1 ----\
              \
Client 2 ------> Singleton.GetInstance() ----> trả về cùng 1 object
              /
Client 3 ----/

code ví dụ c#

public class Singleton
{
    private static Singleton instance;

    private Singleton() { } // chặn new từ bên ngoài

    public static Singleton GetInstance()
    {
        if (instance == null)
        {
            instance = new Singleton();
        }
        return instance;
    }
}
var s1 = Singleton.GetInstance();
var s2 = Singleton.GetInstance();

Console.WriteLine(s1 == s2); // true

👉 Singleton = “Chỉ có 1 object + dùng chung toàn hệ thống”
